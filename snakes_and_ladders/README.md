# Snakes and Ladders (السلم والتعبان)

A classic board game implemented in Python.

## How to Play

1. The game supports 2-4 players
2. Players take turns rolling a dice to move their piece
3. Landing on a ladder moves you up the board
4. Landing on a snake moves you down the board
5. First player to reach position 100 wins!

## How to Run

```bash
python3 snakes_and_ladders.py
```

Follow the prompts to enter the number of players and press Enter when it's your turn to roll the dice.
